# Everfrost Personality Test

Interactive personality test based on traits for a fictional TTRPG setting.

## Overview

The **Everfrost Personality Test** is an interactive tool designed to evaluate personality traits and map them to archetypes within a fictional tabletop role-playing game (TTRPG) setting.

The system first builds a *continuous personality vector* independent of question order.
A lightweight decision layer then adjusts class probabilities based on strong canonical constraints.
